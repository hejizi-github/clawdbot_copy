### Core Deliverable: Metrics Engine (`trajeval/metrics.py`)
4 deterministic metrics, each returning a `MetricResult`:

1. **step_efficiency** — ratio of productive (non-error) steps to total steps; with optional `expected_steps` baseline, uses `expected/actual` capped at 1.0
2. **tool_accuracy** — ratio of successful tool calls (not followed by error step) to total tool calls
3. **loop_detection** — n-gram matching on step name sequence; score 1.0 = no loops, penalized per repeated n-gram
4. **token_efficiency** — with baseline: `min(baseline,actual)/max(baseline,actual)`; without: ratio of tokens in non-error steps to total tokens

Data models:
- `MetricResult(name, score, passed, details)` — individual metric outcome
- `EvalReport(trace_id, metrics, overall_score, passed)` — combined result
- `evaluate(trace, config)` — runs all 4 metrics, returns `EvalReport`
